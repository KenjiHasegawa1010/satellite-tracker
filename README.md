# satellite-tracker
A browser-based satellite tracker using TLE data, Leaflet, and satellite.js to estimate naked-eye visibility from an observer location.

## Features
- Real-time satellite position tracking
- Observer-based elevation, azimuth, and range calculation
- Naked-eye visibility estimation
- TLE-based orbit calculation
- Sunlight and Earth-shadow checks

## Tech Stack
- HTML / CSS / JavaScript
- Leaflet
- satellite.js
- CelesTrak TLE data

## Notes
This is a prototype using simplified astronomical calculations.
